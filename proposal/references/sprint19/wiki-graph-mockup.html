Zoom 機制規格
  • zoom out（< 0.4）：只顯示 L1 module 節點（16px），L2/L3 隱藏
  • 預設 zoom（0.4–1.2）：顯示 L1 module + L2 file/endpoint；L3 method 隱藏
  • zoom in（> 1.2）：全部顯示，L3 method（6px）出現在對應 file 附近；method 標籤 hover 才顯示
  • Zoom 展開圈：虛線圓環標示展開區域，border: 1.5px dashed rgba(245,158,11,0.4)
  • Pulse ring：點擊選中的節點顯示脈衝動畫，border: 2px solid #f59e0b，fade-out scale 動畫
  • Hover 效果：圓點 transform: scale(1.3)，glow 增強到 0 0 16px rgba(color, 0.6)
