## 1. 目標

讓 CodeAtlas 從 2D 平面圖升級為 3D 星圖，視覺衝擊力翻倍。使用者可在 2D/3D 之間一鍵切換——2D 看細節、3D 看全景。3D 模式下節點在空間中漂浮、霓虹光效、粒子沿邊線流動，就像一張「活的星圖」。

**一句話驗收**：切換到 3D 模式 → 旋轉星圖 → hover 節點看路徑亮起 → 「太酷了」。

## 7. 風險

| 風險 | 等級 | 緩解方式 |
|------|------|---------|
| 3d-force-graph 自訂外觀需直接操作 Three.js | 中 | 先用預設外觀跑通，再逐步加霓虹光效 |
| 2D↔3D 切換狀態同步 | 中 | 抽出共享狀態層（selectedNode、searchResults、panelState），兩個渲染器讀同一份 |
| 3D 霓虹光效 GPU 消耗 | 低 | WebGL 天然比 DOM 快。節點 > 200 降低 glow 層級 |
| 面板 DOM overlay 在 3D canvas 上的定位 | 中 | 面板固定在畫面右側（absolute positioning），不跟隨 3D 空間 |
| 新依賴 bundle size 增加 | 低 | 3d-force-graph + three.js ≈ 500KB gzipped。可接受，Phase 4 再優化 |
